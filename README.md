# task-2-Shujat_Ali_Shah

Welcome to the repository for Project 2 of my Virtual Internship as a Data Science and Artificial Intelligence Intern at DecodeLabs. Building upon the rule-based logic from Phase 1, this project transitions into supervised machine learning by training a statistical classification model to predict biological species based on structural measurements.

This repository features an end-to-end Python pipeline using scikit-learn to classify species from the classic Iris dataset, complete with data evaluation, performance metrics, and scatter plot visualizations.
